## FAQ for Marketplace

**Q: Do I need a GitHub account?**
A: Only if you want to push tokens to GitHub. Local download works without GitHub.

**Q: Is my GitHub token secure?**
A: Yes! All credentials are encrypted and stored locally. We never send them anywhere except GitHub API.

**Q: What token types are supported?**
A: Colors, typography, spacing, effects, shadows, gradients, and Figma variables (all types).

**Q: Can I use this in a CI/CD pipeline?**
A: Absolutely! That's exactly what it's designed for. Push tokens to GitHub and let your pipeline handle the rest.

**Q: Is this plugin free?**
A: Yes! Open source under MIT license.

**Q: How fast is token extraction?**
A: ~85ms for typical documents. 96.9% faster than manual export.

**Q: Does it work with large files?**
A: Yes! Dynamic page loading ensures fast startup even with large documents.

**Q: What if I don't want to create a PR?**
A: You can push directly to a branch without creating a PR, or just download locally.
